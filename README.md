# lambda-servicios-linea-proxy
Función AWS Lambda que actúa como capa proxy para Servicios Línea, gestionando el enrutamiento, validación e integración con servicios backend.

Este repositorio contiene los siguientes servicios:

Consultas a Siebel:
- addonsByAccount
- getInfoAccount
- getPendingOrders

Consultas hacia SOA:
- impulseUpgrade
